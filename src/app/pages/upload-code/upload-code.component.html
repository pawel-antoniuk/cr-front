About CodeRunner

info_outline
The application allows you to test and compare implementations of algorithms. The implementation code is loaded in one of the available programming languages. The code is assigned as a solution to a given a task. Each task consists of test cases that define what result should be returned by the algorithm for a given input. CodeRunner provides an analysis module that allows you to analyze the results stored by the application, which allows you to choose the optimal solution.
